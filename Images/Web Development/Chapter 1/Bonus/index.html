An assortment of cool HTML and CSS tags

Here's a few HTML tags:

  1. h# - Heading 1-6
  2. ol - Ordered list
  3. ul - Unordered list
  4. a - Anchor Tag
  5. p - Paragraph
  6. div - Division
  7. img - Image
  8. span - Style a part of text
  9. br - Break
  10. href - Hypertext reference

More about them:

    1. <h#> or Heading 1-6 Tags

- Headings serves as the titles for your webpage. There's is a max of 6 you can use each getting smaller and smaller in the emphasis and font size the higher you go. H1 being the largest and most important and H6 being the smallest and least important of all.

Example:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
Output

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6
    2. <ol> or Ordered List Tag

- Ordered lists is a more specific way of organizing your content through using a number system using a line element.

Example:

<ol>
	<li>Apple</li>
	<li>Orange</li>
	<li>Banana</li>
	<li>Strawberry</li>
</ol>
Output:
  1. Apple
  2. Orange
  3. Banana
  4. Strawberry
    3. <ul> or Unordered List Tag

- Unordered lists are generally similar to the one previous, Ordered lists. The main difference is that it uses bullet points instead of a numerics.

Example:

				
<ul>
	<li>Apple</li>
	<li>Orange</li>
	<li>Banana</li>
	<li>Strawberry</li>
</ul>
Output:
  • Apple
  • Orange
  • Banana
  • Strawberry
    4. <a> or Anchor Tag

The Anchor Tag is a way for developers to add hyperlinks to their websites to redirect users to another site. Like references.

Example:

<a href="Study labs"> https://portal.ftcglobal.ae/login/index.php</a>
Output:
https://portal.ftcglobal.ae/login/index.php
    5. <p> or Paragraphs

The paragraph tag is the probably most simple tag in html. It's job is to hold smal to big lines of text and display it on the site.

Example:

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
Output:

Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

    6. <div> or Division Tag

The Division tag is to separate a website's content into more managable blocks which makes editing, sorting, and keeping track of things a whole lot easier. These tags are also very usefull when editing specific sections in CSS.

Example:

<div=class"Opening Paragraph">
	<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
</div>
    7. <img> or Image Tag

The image tag is paired with multiple other tags that lets the developer choose specific pictures from specific sources ranging from links to local files.

Example:

<img src="https://th.bing.com/th/id/OIP.r8V3TVK8707ofTYj027DwQHaE8?w=245&h=180&c=7&r=0&o=5&pid=1.7">
Output:
    8. <span> or Span Tag

The Span tag is a general-purpose inline container for content and inline elements. When no other semantic element is available, it is used to group elements for stylistic purposes (by using the class or id properties).

Example:

<span style="color:blue;font-weight:strong">BathSpaUniversity</span>
Output:
BathSpaUniversity
    9. <br> or Break Tag

The Break Tag is used to split long lines of text into the next line instead of continously going to the right or left of the screen.

Example:

<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.<br>
 Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.<br>
Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident,<br>
sunt in culpa qui officia deserunt mollit anim id est laborum.</p>
Output:

Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident,
sunt in culpa qui officia deserunt mollit anim id est laborum.

    10. <href> or Hypertext Reference Tag

The href tag is used after the anchor tag, it's responsible for making external links be accessible in your website.

Example:

<a href="Study labs"> https://portal.ftcglobal.ae/login/index.php</a>
Output:
https://portal.ftcglobal.ae/login/index.php

Here's a few CSS tags:

  1. background-clip
  2. background-origin
  3. background-size
  4. align-content
  5. flex
  6. order
  7. font-family
  8. border-image

More about them:

    1. background-clip

- Accurately controls how far the color and image is from the element.

    2. background-origin

- Accurately control where the background image goes to.

    3. background-size

- For resizing the image.

    4. align-content

- Tries to align a "flex" container's lines vertically.

    5. flex

- Groups content for a flexible length.

    6. order

- Specifically arranges the order of a flexible item in a flex or grid container.

    7. font-family

- Allows for custom or premade font styles for use in the website.

    8. border-image

- Allows for the use of an image as a border to an element.
